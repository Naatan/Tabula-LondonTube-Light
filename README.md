This is a color preset for the [Tabula Rasa] skin. It's based on the LondonTube-Light color scheme.

## Installation and Usage

 * install [Tabula Rasa]
 * Go to Preferences > Appearance
 * Select the "Tabula Rasa" skin and Iconset
 * Hit the "Import" button under "Skin Colors"
 * Select the file you downloaded here and press Ok
 
For best results you should also switch to the LondonTube-Light color scheme under Preferences > Color Scheme.

   [Tabula Rasa]: http://komodoide.com/packages/skins/tabula-rasa/
